Biodiesel Production Plant

Design and implement a small-scale biodiesel production system using waste cooking oil.

2-3 months

Project Overview

This project involves designing and implementing a small-scale biodiesel production system that converts waste cooking oil into usable biodiesel fuel through transesterification.

Implementation Steps
  • Collect and preprocess waste cooking oil
  • Design reactor system for transesterification
  • Implement purification and separation processes
  • Test fuel quality and properties
  • Optimize production parameters
Required Components
  • Reactor vessel with temperature control
  • Methanol and catalyst storage
  • Separation and purification equipment
  • Testing equipment for fuel properties
Resources
  • Biodiesel Production Handbook
  • Transesterification Process Guide
  • Fuel Quality Testing Standards
